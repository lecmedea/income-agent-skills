# Income Agent Skills (Grok / Codex)

Набор skills для автономного поиска заказов и маркетинга РФ 2026.

| Skill | Описание |
|-------|----------|
| [income-agent](./income-agent/) | Hunt → Propose → Deliver → Report |
| [income-agent-schedule](./income-agent-schedule/) | Расписание без лишних токенов |
| [income-agent-autonomy](./income-agent-autonomy/) | Легальная автономия заработка |
| [russian-marketing-2026](./russian-marketing-2026/) | Тренды Чижова 2026 |

## Установка в Grok

```bash
cp -R income-agent ~/.grok/skills/
cp -R income-agent-schedule ~/.grok/skills/
cp -R income-agent-autonomy ~/.grok/skills/
cp -R russian-marketing-2026 ~/.grok/skills/
```
